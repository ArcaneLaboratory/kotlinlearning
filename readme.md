# Command Line Calculator

A (very) simple calculator on the command line built in Kotlin.

## Instructions for Build and Use

Steps to build and/or run the software:

1. Clone from git using IntelliJ
2. Build using run button

Instructions for using the software:

1. Enter first operand (operands must be numbers)
2. Enter second operand
3. Enter operation
4. View result, type quit to quit or anything else to run additional operations
5. Observe that previous operation history is displayed after each operation runs

## Development Environment

To recreate the development environment, you need the following software and/or libraries with the specified versions:

* IntelliJ IDEA (any version)
* Java (version 8 or later)
